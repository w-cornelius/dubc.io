Gmail Signature Setup Guide

Follow these three steps to get your professional signature live.

1

Host Your Images

You need a permanent place on the web for your PNGs to live.

  1. Go to GitHub and create a new Public Repository (e.g., named email-assets).
  2. Upload the 7 PNG files you downloaded from the Asset Manager.
  3. Go to Settings > Pages.
  4. Under Build and deployment, change "Source" to Deploy from a branch.
  5. Select main (or master) and / (root), then click Save.
  6. Wait about 1-2 minutes. Your images are now hosted!
3

Install in Gmail

Gmail's normal editor breaks complex HTML. Use this trick:

  1. Go to Gmail Settings > See all settings.
  2. Scroll down to the Signature section.
  3. Type a placeholder word like "HERE" in the signature box.
  4. Right-click "HERE" and select Inspect (or Inspect Element).
  5. A developer panel will open highlighting a line of code. Right-click that highlighted line and select Edit as HTML.
  6. Paste the code generated in Step 2.
  7. Click anywhere outside the box to apply changes. You should see your signature appear!
  8. Scroll to the bottom of Gmail and click Save Changes.
