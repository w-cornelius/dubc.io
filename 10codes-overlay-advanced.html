Code Red
A Code Red situation is where the suspect is putting the officer's or civilian's life in danger at which time lethal force may be used.
Code Amber
A Code Amber situation is where the suspect is armed or dangerous but lethal force is not required. In this case, non-lethal force can be used.
  • If the suspect is using a pistol, Police officers can use their pistol.
  • If the suspect has a shotgun, then you can use their pistol.
  • If the suspect is using an MP5, Police can use their MP5.
  • If a police vehicle has been stolen then it is automatically code red to disable that vehicle only.
  • If a vehicle actively opens up fire on officers, again this is code red and you are to disable the vehicle.
  • If a suspect has NO hostage and is pointing a gun at multiple officers they are code amber, you are to warn them 3-4 times to drop the weapon and if they don't comply the highest-ranking officer can make the call to code red, and put the suspect down. (IF THE SUSPECT HAS A HOSTAGE YOU ARE TO NEVER GO CODE RED)
  • If you are solo and pursuing a suspect on foot and they have a gun in hand and point it towards you, you can instant code RED and open fire.
Impound Categories and Fines

Normal Impound (Traffic or Parking Violations)

  • Fine / Time: $500 / Retrieval By Owner
  • Procedure: Issue a citation if applicable. Notify the driver (if present) and provide them with impoundment details. Notify SADOT or Impound the Vehicle if SADOT is not available.

Evading Arrest or Pursuit

  • Fine / Time: $2,500 / 1 Hour
  • Procedure: Ensure the pursuit is logged in the system, including timestamps. Clearly document the evasion in the report and link it to the impoundment record. Notify SADOT or Impound the Vehicle if SADOT is not available, following the apprehension of the driver or abandonment of the vehicle.

Criminal Activity (Robberies or Heists)

  • Fine / Time: $5,000 / 4 Hours
  • Procedure: Confirm the vehicle's involvement in the criminal act through evidence or witness statements. Include crime scene evidence and photos in the impound report. Add Pictures of the Vehicle to the Case report in the MDT.

Shootouts with LSPD

  • Fine / Time: $10,000 / 6 Hours
  • Procedure: Ensure officer safety and secure the scene. Complete a critical incident report in addition to the impound record, Including Pictures. Notify SADOT or Impound the Vehicle if SADOT is not available.
